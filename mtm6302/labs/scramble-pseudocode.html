# Scramble Pseudocode Participation

A GitHub Classroom Assignment

This assignment must be accepted on GitHub Classroomopen in new window.

# Objective

For this exercise, create a file containing pseudocode, which will serve as a guideline for completing Scramble.

# Requirements

  1. Clone the Scramble repository from Github.
  2. Inside the scramble.js file, write the pseudocode to outline the logical structure of the game. It should include the necessary objects, arrays and functions.

# Submission

  1. Create a commit with the message "Creates the Scramble pseudocode"
  2. Push all commits to GitHub.
  3. Submit the URL to your GitHub repository to the Scramble Pseudocode assignment on BrightSpace.
