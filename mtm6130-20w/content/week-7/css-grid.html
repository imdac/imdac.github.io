              From MDN: CSS Grid Layout excels at dividing a page into major regions or defining the relationship in terms of size, position, and layer, between parts of a control built from HTML primitives.

              # Grid Key features

              • Grid Items - Only direct decedents of the grid container. Can contain a grid inside a grid
              • True Grid - no masonry pattern
              • Content Stacking - Placed using HTML Source order further down the html, higher up stacked. Can be changed using z-index property
              • Content Order - any item can be placed anywhere in the grid so visual order can be changed using grid without worrying about HTML. Make sure it is accessible, proper HTML markup and layout.

              # Defining a Grid Container

              This YouTube video was created by Steve Griffith.

              Grid items are automatically placed in the grid by default, unless we tell each item otherwise using grid-column and grid-row properties

              # Fraction Units

              Grid has a new unit to be used using fr, fraction unit

              Fraction Units(fr) 1fr 1fr like 50% 50% - it works like the flex grow property

              # Minmax Function

              Similar to min-width and max-width, 1fr minmax(10em, 20em) 1fr

              # Repeat function

              We can use the repeat function to repeat columns, rather than writing 1fr 1fr 2fr we can write repeat(2, 1fr) 2fr

              more advanced way of using repeat is repeat(4, 1fr 10px) this will create 1fr 10px 1fr 10px 1fr 10px 1fr 10px

              # Grid items placement and span

              Using grid-column and grid-row properties on the grid items we can define the starting and ending column or row line for the item.

              Span keyword is used with grid-column or grid-row to tell the element span a certain number or columns or rows without the need specify the start and end line.

              Implicit Lines - when we need to place content outside the predefined grid lines browser automatically creates lines called implicit lines.

              # Naming the Grid Lines

              You can give custom names to the grid lines, using [] in the template-columns property of the grid container

              # Grid Area and Grid Gap

              • Area is applied to grid container using grid-template-areas property.
              • We use text names to create a grid map and assign grid area names to individual cells.
              • We assign grid items to different grid areas by using the grid-area property on the grid items and assigning it the label of the area we defined.
